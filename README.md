# cypress-basico-v2

Exemplo do projeto basico da escola Talking about testing

## Pré requisitos

É necessário ter Node.js e npm instalados para executar este projeto.

> Para este projeto foi utilizado as versões `v16.13.2` e `8.1.2` do Node.js e npm, respectivamente. Eu sugiro que você use as mesmas versões ou versões posteriores.

## Instalações

Executar `npm install` (ou `npm i` para a versão curta) para instalar as dependências de desenvolvimento.

## Testes

Voce pode rodas os testes simulando desktop ou mobile.

### Desktop

Execute `npm test` para executar o teste no modo headless em uma janela de visualização de desktop

ou, execute `npm run cy:open` para abrir o Cypress no modo interativo em uma viewport de desktop.

### Mobile

Execute `npm test:mobile` para executar o teste no modo headless em uma janela de visualização do dispositivo móvel.

ou, execute `npm run cy:open:mobile` para abrir o Cypress no modo interativo em uma janela de visualização do dispositivo móvel.
